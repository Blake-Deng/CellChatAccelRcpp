# Paper

This directory contains the Bioinformatics Application Note manuscript source and submission-facing materials.

## Main Files

| file or directory | purpose |
| --- | --- |
| `main.tex` | LaTeX manuscript source using the OUP authoring template |
| `main.pdf` | compiled manuscript PDF |
| `references.bib` | bibliography |
| `figures/1.pdf` | main six-panel manuscript figure |
| `figures/Fig01_*` to `figures/Fig06_*` | individual panel/source figure files |
| `Supplementary_Table_S1.csv` | dataset accession table for Data availability |
| `Supplementary_Table_S1.tsv` | TSV copy of Supplementary Table S1 |
| `oup-authoring-template.cls`, `oup-plain.bst`, `oup-abbrvnat.bst` | journal template files |
